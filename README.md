# Aplicacion de comandas

Una aplicacion que funciona a tiempo real para que desde un dispositivo se pueda mandar una comanda y desde otro pueda verse el pedido. Ademas tambien se pueden cargar productos a un menu.



## Autores

- [Pedro Perez Mochetti](https://github.com/elmoche150)


## FAQ

#### ¿Porque una aplicacion asi?

Basicamente, en mi familia hay un emprendimiento de un bar, en donde cada vez que se necesita llevar una comanda tiene que ser fisicamente y subiendo escaleras. De esta manera se soluciona ese problema
